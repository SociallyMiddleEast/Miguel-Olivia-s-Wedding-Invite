# 💍 Miguel & Olivia — Wedding Website

## Folder Structure

```
wedding-miguel-olivia/
├── index.html          ← Main website file
├── images/             ← Place ALL your images here
│   ├── hero.jpg        ← Big hero photo of you & Olivia
│   ├── story-1.jpg     ← Story section photos
│   ├── story-2.jpg
│   ├── story-3.jpg
│   ├── story-4.jpg
│   ├── ceremony.jpg    ← Church photo for ceremony section
│   └── restaurant.jpg  ← Restaurant photo
├── music/              ← Place your background song here
│   └── song.mp3        ← Rename your song to song.mp3
├── css/
│   └── (auto-generated styles embedded in index.html)
└── js/
    └── (auto-generated scripts embedded in index.html)
```

## Google Sheets Setup

### Sheet 1 — Guest List
- Column A: Guest Full Name
- Column B: Custom Invite Link (auto-generated by the website)
- Columns C–J: Any extra info you want

### Sheet 2 — RSVP / Reservations
- This is where guests confirm attendance
- Columns: Name, Attending (Yes/No), Number of Guests, Message, Timestamp

### How to connect Google Sheets:
1. Open your Google Sheet
2. Go to **File → Share → Publish to web**
3. Choose **CSV format**, select Sheet 1, click Publish
4. Copy the URL and paste it in `index.html` where it says `YOUR_GOOGLE_SHEET_CSV_URL`
5. For RSVP form submissions, use the Google Form linked to Sheet 2 and paste the form action URL where it says `YOUR_GOOGLE_FORM_ACTION_URL`

## Hosting on GitHub Pages

1. Create a new GitHub repository named `miguel-olivia-wedding` (or any name)
2. Upload ALL files keeping the folder structure
3. Go to **Settings → Pages**
4. Set source to **main branch / root**
5. Your site will be live at: `https://yourusername.github.io/miguel-olivia-wedding`
